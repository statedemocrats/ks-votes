Kansas Voting History

Use the controls in the upper right corner of the map to toggle map layers on/off.

Precinct boundaries are from 2012 (originally 2010 census), the most recent state-wide shapefile available. As a result, when viewing precinct results from more recent elections, the color yellow is used to indicate a precinct that has changed boundary and/or name since 2012 and can no longer be neatly aligned with the precinct at election time. See list of more recent shapefiles without election results.
